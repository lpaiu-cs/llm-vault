# 05_Inbox — 미처리 인입 (Unprocessed Intake)

아직 판단·구조화하지 않은 미가공 자료를 일단 떨어뜨리는 곳입니다. 성격에 맞는 하위
폴더에 넣고, 안정적으로 증거 가치가 있으면 [[Ingest Policy]]에 따라 `06_Raw/`로 이관합니다.

| 하위 폴더 | 용도 |
|-----------|------|
| `chats/` | 처리 전 대화 로그 |
| `notes/` | 잡다한 수기 메모 (승격 불필요할 수 있음) |
| `screenshots/` | 스크린샷 |
| `links/` | URL, 북마크 |
| `code-logs/` | 코드 조각, 에러 로그 |
| `admin-records/` | 행정 관련 미처리 자료 |

> 이 계층은 **인덱싱에서 제외**됩니다(휘발성). 그래프 node가 되지 않습니다.
> 관련 정책: [[Ingest Policy]], [[Source Policy]], [[Second Brain Operating Model]].
